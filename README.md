### 三、功能描述
系统分为（身份）：普通用户、体检员、后勤人员、护工、管理员。

1. 登录
2. 注册
3. 修改密码
4. 个人中心
5. 房间信息
6. 房间入住信息
7. 反馈信息管理
8. 留言信息
9. 老人信息
10. 公告信息
11. 物资申请管理

**体检员功能：**

9. 留言信息
10. 公告信息
11. 物资申请信息


**后勤人员功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 房间信息
6. 房间入住信息
7. 反馈信息
8. 老人信息
9. 留言信息
10. 公告信息

**护工功能：**
